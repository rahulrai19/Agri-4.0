# Agri 4.0 Web App

## How to Run

### Option 1: One-Click Script
Double-click `run.bat` in this folder.

### Option 2: Manual
1. Install dependencies:
   ```powershell
   npm install
   ```
2. Run server:
   ```powershell
   npm run dev
   ```

## Build for Production
```powershell
npm run build
```
